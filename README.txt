
/////////////General Information/////////////
Goal:
 RADIUS packet encoding/decoding library for NODE.JS servers. 

With node-radius server you can easily decode received packets (get the username and password), encode packets you are 
sending and prepare responses to receive packets. It supports RADIUS authentication and RADIUS accounting packets.

/////////////Needs for Our Project///////////
* Authentication needed
*Read Username and Password, if the user does not exist in our database do not send the packet 
* IP Range for the incoming user to be able to say anything outside of range is rejected
